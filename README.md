**Todo App** — это полноценное веб-приложение для управления задачами, построенное на современном стеке технологий. Проект демонстрирует чистую **слоистую архитектуру** и полностью контейнеризирован с помощью Docker.
## 🚀 Особенности проекта

- ✅ Создание, чтение, обновление и удаление задач (CRUD)
- 🐳 Полная контейнеризация всех сервисов через Docker Compose
- 🏗️ Чистая **слоистая архитектура (Layered Architecture)** на бэкенде
- 🔗 Взаимодействие компонентов через изолированную Docker-сеть
- 📦 Постоянное хранение данных через Docker Volume

---

## 🛠️ Технологический стек

### Фронтенд
| Технология | Назначение |
|------------|------------|
| React 18 | UI библиотека |
| Node.js | Среда выполнения |
| npm | Пакетный менеджер |

### Бэкенд
| Технология | Назначение |
|------------|------------|
| FastAPI | Веб-фреймворк (Python) |
| SQLAlchemy | ORM |
| Alembic | Миграции БД |
| Pydantic | Валидация данных |

### База данных и инфраструктура
| Технология | Назначение |
|------------|------------|
| PostgreSQL 17 | Реляционная БД |
| Docker / Compose | Контейнеризация |
| Docker Networks | Изоляция сервисов |

## Запуск проекта

```bash
# 1. Клонировать репозиторий
git clone https://github.com/yourusername/todo-app.git
cd todo-app

# 2. Создать файл .env с переменными окружения
DATABASE_URL=postgresql+psycopg://postgres:admin@postgres:5432/postgres
ALLOWED_ORIGINS=http://localhost:3000
EOF

# 3. Запустить все сервисы
docker-compose up -d

# 4. Проверить статус контейнеров
docker-compose ps
